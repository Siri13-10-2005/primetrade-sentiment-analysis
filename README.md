# Primetrade-sentiment-analysis
# Bitcoin Market Sentiment and Trader Performance Analysis

## Objective

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader profitability using Hyperliquid historical trading data.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Key Findings

* Extreme Greed produced the highest average profit per trade.
* Fear periods generated the highest cumulative profit.
* BUY trades performed best during Fear conditions.
* SELL trades performed best during Extreme Greed conditions.
* SOL and ETH showed strong average profitability.
* Top traders exhibited different performance patterns across sentiment regimes.
* Trade size showed only a weak positive correlation with profitability.

## Conclusion

The analysis demonstrates a measurable relationship between market sentiment and trader performance. Different market regimes favor different trading behaviors, highlighting the importance of sentiment-aware trading strategies.
